# Todo List App

A simple todo list app built with HTML, CSS and JavaScript, using Firebase Firestore and Realtime Database.

## Features
- Add and delete todos
- Mark todos as completed
- Syncs in real-time across different users/tabs

## How to run

1. Clone the repo
2. Open index.html in your browser - thats it, no installs needed

## Firebase
- Firestore is used to store the todos
- Realtime Database is used to sync changes live across users

## Notes
Built this as a straightforward todo app, kept the code simple and readable. 
Used the compat version of Firebase SDK because its cleaner to work with for a project this size.
